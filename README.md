# Solid-State-Tesla-Coil
--------------!! WARNING !!-------------------

The Magnetic feild generated by this tesla coil may cause damage to electronic devices in a radius of 20cm around the coils

------------ Components used ----------------

- MOSFET IRF540.
- Cap 1uF.
- Primary coil : 1 spin.
- Secondary coil : 500 spin / diametre 1.9mm.
- 12v/3A Power supply.
